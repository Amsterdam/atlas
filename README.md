Atlas
=====

Requirements
- NPM
- Bower
- Grunt

Installation
- npm install
- grunt
- Open the server at http://localhost:8000/

Other useful commands
- npm test
- grunt build-release

Techniques used
- AngularJS
- Redux
- Leaflet
- D3

Conventions used
- John Papa Angular styleguide (https://github.com/johnpapa/angular-styleguide/tree/master/a1)
- EditorConfig (http://editorconfig.org/)
- BEM
